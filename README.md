📌 Overview
This project is a high-fidelity Task Card UI built as part of a frontend assessment. It focuses on semantic HTML, accessibility, and dynamic time-based updates.

⚙️ Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Dynamic due date and time remaining
- Auto-refresh every 60 seconds
- Semantic HTML structure
- Fully mapped `data-testid` attributes for testing

🧱 Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
